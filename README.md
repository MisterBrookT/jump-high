# Jump High

A terminal Doodle-Jump-style game built with Rust + ratatui.

![screenshot](screenshot.png)

## Install

```bash
cargo install --path .
```

## Controls

| Key | Action |
|-----|--------|
| ← → | Move left/right |
| q / Esc | Quit |

Character auto-jumps when landing on a platform. Score = max height reached.
